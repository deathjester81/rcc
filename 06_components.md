## Components

### Status Badge
- Completed (green)
- In Progress (yellow)
- Available (gray)

### Phase Card
- Phase name
- Short description
- Status badge
- Optional "Kontakt aufnehmen" button

### Contact Button
- Label: "Kontakt aufnehmen"
- Link: https://r-c-c.ch/termin/
- Only visible if status = Available

### Download List
- Document name
- File type icon
- Download action
