# RCC Customer Journey Mockup

Read-only Mockup zur Visualisierung der RCC Customer Journey pro Kunde.

## Tech-Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS

## Installation

```bash
npm install
```

## Entwicklung

```bash
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Projektstruktur

- `/app` - Next.js App Router Seiten
- `/components` - React Komponenten
- `/data` - Mock-Daten für Kunden und Phasen
- `/types` - TypeScript Interfaces

## Features

- Customer List mit 4 Beispiel-Kunden
- Customer Project Workspace mit Roadmap Timeline
- Phase 2 Details mit 4 Elementen (Prozessboard, Umsetzungs-Session, Kernteam-Review, Gruppencoaching)
- Gruppencoaching Subsite mit Dokumenten-Downloads
- RCC Branding (hellblau/weiß)
